快速开始

TG-Image 是一个简单易用的免费图床服务，让您轻松上传、管理和分享图片。跟随以下步骤快速上手：

1

注册账户

访问登录页面，点击"注册账户"创建您的专属账户。注册后您可以享受更多高级功能。

2

上传图片

在首页拖拽图片到上传区域，或点击选择文件。支持 JPG、PNG、GIF、WebP 等多种格式。

3

获取链接

上传完成后，系统会自动生成图片链接。您可以复制链接在任何地方使用。

4

管理图片

在"我的图片"页面管理您的所有上传内容，支持编辑、删除、收藏等操作。

常见问题
支持哪些图片格式？

我们支持所有主流的图片格式，包括：JPG/JPEG、PNG、GIF、WebP、BMP、SVG 等。建议使用 JPG 或 PNG 格式以获得最佳兼容性。

单个文件大小限制是多少？

免费用户单个文件最大支持 10MB，注册用户可上传最大 50MB 的文件。如果您需要上传更大的文件，请联系客服。

图片会被压缩吗？

我们提供三种图片质量选项：高质量（原始）、中等质量（推荐）、低质量（节省空间）。您可以在设置中调整默认质量，也可以在上传时手动选择。

图片链接会过期吗？

不会！所有上传的图片链接都是永久有效的，除非您主动删除。我们承诺为用户提供稳定可靠的图片存储服务。

如何删除已上传的图片？

登录后在"我的图片"页面，点击图片的"删除"按钮即可删除。注意：删除操作不可撤销，请谨慎操作。

是否支持批量操作？

是的！您可以选择多张图片进行批量删除、复制链接、添加标签等操作。这大大提高了管理效率。

API 文档

TG-Image 提供完整的 RESTful API，方便开发者集成到自己的应用中。以下是主要接口说明：

基础信息

API 基础地址： https://tg-image.com/api

认证方式： Bearer Token

请求格式： JSON

响应格式： JSON

上传图片

POST /upload

Content-Type: multipart/form-data

{
  "file": "图片文件",
  "quality": "high|medium|low",
  "public": true|false
}

获取图片列表

GET /images

参数： page, limit, sort, filter

删除图片

DELETE /images/{id}

认证： 需要有效的认证令牌

更多详细的 API 文档请访问我们的 开发者文档。
